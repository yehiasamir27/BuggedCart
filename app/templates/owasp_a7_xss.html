Types of XSS

  • Reflected XSS: Payload in URL/form, reflected back immediately
  • Stored XSS: Payload stored in database, affects all visitors
  • DOM-based XSS: Client-side JavaScript manipulation

Impact:

  • Session hijacking (steal cookies)
  • Credential theft
  • Keylogging
  • Phishing attacks
  • Malware distribution

Prevention:

  • Escape HTML output (use template auto-escaping)
  • Content Security Policy (CSP)
  • HTTPOnly cookies
  • Input validation and sanitization
