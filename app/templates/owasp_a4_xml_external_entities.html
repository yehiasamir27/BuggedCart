How XXE Works

XXE attacks exploit XML parsers that process external entity references.

Attack Vector:

<!DOCTYPE foo [<!ENTITY xxe SYSTEM "file:///etc/passwd">]>

Impact:

  • Read local files on server
  • Server-Side Request Forgery (SSRF)
  • Denial of Service (billion laughs attack)
  • Port scanning internal network

Prevention:

  • Disable external entities in XML parser
  • Use defusedxml library in Python
  • Validate and sanitize XML input
  • Use simpler data formats (JSON) when possible
