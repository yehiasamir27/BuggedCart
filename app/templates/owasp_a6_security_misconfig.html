Common Misconfigurations

  • Debug Mode in Production: Exposes stack traces, source code, and allows code execution
  • Default Credentials: admin/admin, root/root, etc.
  • Verbose Error Messages: Reveal database schema, file paths
  • Directory Listing Enabled: Shows file structure
  • Unnecessary Features Enabled: Unused endpoints, services
  • Missing Security Headers: X-Frame-Options, CSP, etc.

Prevention:

  • Disable debug mode in production
  • Use security scanners (nikto, nmap)
  • Configure proper error handling
  • Remove default accounts
  • Implement security headers
  • Regular security audits
