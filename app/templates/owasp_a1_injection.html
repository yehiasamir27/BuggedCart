How SQL Injection Works

SQL Injection occurs when user input is directly concatenated into SQL queries without sanitization.

Vulnerable Code:

sql = "SELECT * FROM product WHERE name LIKE '%" + user_input + "%'"

Safe Code (Parameterized):

cursor.execute("SELECT * FROM product WHERE name LIKE ?", (f'%{user_input}%',))
